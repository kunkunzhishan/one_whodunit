# 零点灯室

对话驱动的 AI 推理游戏。玩家扮演侦探，在一座孤岛上审讯 6 名嫌疑人，还原 15 年前的旧案真相。

## 快速开始

**1. 安装依赖**
```bash
npm install
```

**2. 配置 API**

新建 `.env` 文件：
```env
OPENAI_BASE_URL=xxx
OPENAI_API_KEY=你的key
OPENAI_MODEL=xxx
PORT=3000
```

**3. 启动**
```bash
npm start
# 或开发模式（自动重启）
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)

## 项目结构

```

├── src/
│   ├── game-engine.js     # 游戏核心逻辑

│   └── memory/            # 角色记忆存档（.json，运行时生成）

```

## 记忆系统

每个角色有独立记忆，随对话动态更新：
- **信任度**（0-100）：随玩家态度升降
- **情绪**：跟着对话实时变化
- **对话摘要**：每 2 轮自动分析存档
- **玩家印象标签**：角色对玩家的判断

记忆文件存在 `data/memory/` 下，新游戏时重置。

---

## 游戏流程

共 5 个阶段，每个阶段推进后：
- 解锁新线索
- 可能有角色死亡
- 角色记忆对话历史保留（不会因换阶段忘记）

提交推理后由 AI 裁判判断是否接近真相，答对触发结局 A，全程未答对触发结局 B。
